### Informações de Desenvolvimento
O Projeto Noteblock ( sem nome ainda definido.) É um simples bloco de notas escrito em linguagem Julia.

# Versões
Alpha 1.0 ::
Adição dos primeiros módulos Backend do projeto.

# Suporte
O projeto tem suporte em qualquer compilador de linguagem julia na Alpha 1.0, isso por ser apenas um Backend
que roda em um terminal.

### Recomendações
Por minha parte, é sempre recomendado rodar qualquer código no Linux ou BSD idependente da linguagem, isto é,
por menos riscos de quebrar o seu sistema, e por segurança.
Também é recomendado usar Compiladores e Terminais confiáveis pra rodar o código! Usar algo de origem desocnhecida
pode afetar sua segurança e injetar arquivos maliciosos no seu sistema.
